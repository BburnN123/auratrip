# Auratrip

Come up with the architecture for a trip planning app for a hackathon.

# Requirements

1. Use Bright Data to scrape data from social media platforms e.g. Instagram, TikTok
2. Gather realtime weather information from real user posts
3. Generate a trip itinerary based on information that is collected from scraping e.g. user inputs "Bangkok" -> generate a trip itinerary for Bangkok on the actual day

# Required AI stack

​Bright Data: Give your assistant the ability to browse the web. Build custom web-scraping "skills" into your agents to fetch real-time data, monitor trends, or aggregate research automatically using reliable, large-scale infrastructure.

​Daytona: an open-source, secure infrastructure platform providing fast, isolated, and stateful composable computers (sandboxes) for AI agents to safely run, test, and develop code. Each hackathon participant will get 100 USD worth of Daytona credits to build.

​Kimi AI: Providing the kimi-k2.6 API, featuring SOTA coding, cost-efficient context caching, and native agent swarm execution to empower developers building complex, multi-step hackathon solutions.

​Nosana: Supercharge your compute with Nosana's decentralized GPUs to run specialized AI models. Use them to run tasks, process heavy data workloads, or host custom local models for your assistant affordably.

​SenseNova U1: an open-source, native multimodal AI model that seamlessly unifies text understanding, reasoning, and high-fidelity image generation within a single, end-to-end architecture. While its SenseNova U1 Skills provide modular expert capabilities to automate complex Excel analysis, PowerPoint creation, and deep research.

​Terminal 3's Agent Dev Kit: allows developers to equip AI agents with verifiable identity to act and transact on behalf of users without compromising sensitive data. Built on the Terminal 3 Network’s hardware-secured TEEs, it provides autonomous AI workflows with the trust and security guarantees they need.

​TokenRouter: a unified routing platform offering seamless access to the top AI models worldwide. It delivers faster, better, and cheaper performance via smart caching.

​VideoDB: Data infrastructure for video, built for machines and agents - Turn camera feeds, live streams, files, and recordings into searchable context, real-time alerts, clips, and agent workflows.

Use all products in the stack as much as possible as this impacts judging.
